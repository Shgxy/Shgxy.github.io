选择器

作用：选择页面上的某一个或者某一类元素。

🧐基本选择器

标签选择器：“全部”

会选择到页面上所有的这个标签的元素。

<style>
    h1{
      color: green;
      background-color: darkcyan;
      border-radius: 10px;
    }
    p{
      color: brown;
    }
</style>
类选择器：“部分”

仅同一类名的类会更改样式。

如 class 名为”shgxy”。

<style>
    .shgxy{
      color:black;
    }
</style>
id选择器：“唯一”

id必须保证全局唯一。

如 id 名为”blog”。

<style>
    #blog{
      color: green;
    }
</style>
基本选择器小结
> 不遵循就近原则，固定的 > > id选择器 > 类选择器 > 标签选择器

层次选择器

后代选择器

在某个元素后面。

如 p 后面。

/*后代选择器*/
body p{
	background-color: green;
}
子选择器

一代。衍生的不算。

/*子选择器*/
body > p{
  background-color: green;
}
相邻兄弟选择器

同辈。选择定义有 active 的下一个标签。只向下选择一个。

.active + p{
	background-color: green;
}
通用兄弟选择器

当前选中元素的向下的所有兄弟元素。

.active ~ p{
	background-color: green;
}

结构 伪类选择器

格式 p1
p2
p3
  - li1
  - li2
  - li3

ul下的第一个li：

<style>
  ul li:first-child{
    background-color: green;
  }
</style>

ul下最后一个li：

<style>
  ul li:last-child{
    background-color: green;
  }
</style>

第一个p标签：

<style>
  /*选中p1：定位到父元素，选择当前的第一个元素
    选择当前p元素的父级元素，选中父级元素的第一个，并且是当前元素才生效。
    */
  p:nth-child(1){
    background-color: green;
  }
</style>
<style>
  /*选择父元素下p元素的第二个*/
  p:nth-of-type(1){
    background-color: green;
  }
</style>

🧐属性选择器

>相当于是 id + class 二者结合。 
a[id]{
    /*选中a标签中带有id属性的*/
}
a[id=first]{
    /*选中a标签中id属性为first的*/
}
a[class*=“links”]{
    /*选中a标签中class属性中含有links的
    *=	：是包含这个links的意思
    =	：是绝对一致的意思
    */
}
a[href^=“https”]{
    /*选中a标签中href属性以https开头的
    ^=：以这个开头
    */
}
a[href$=jpg]{
    /*选中a标签中href属性以jpg结尾的
    $=：以这个结尾
    */
}

数组

数组中可以包含任意的数据类型。

  • arr.length获取arr数组的长度，但如果给arr.length赋值，数组会发生变化
    • 赋大值，数组中补上undefined
    • 赋小值，数组截断，后面的元素丢失
  • arr.indexOf(0)返回第一个字符出现0的下标索引，若不含0,返回-1
  • slice()，类似字符串中的substring()
  • push()压入尾部，pop()尾部弹出一个元素并返回
  • unshift()头部添加，shift()头部弹出并返回
  • sort()根据字符排序
  • 🔴reverse() 反转
  • concat([])，返回拼接后的数组，原数组不修改
  • join()，使用特定字符串拼接，类似python

arguments

通过js预置的arguments，可以获得传入函数的所有参数。

function abs (x){
    console.log("x:"+x);
    for (arg of arguments){
        console.log("arguments:"+arg);
    }
    if(typeof (x)!="number"){
        throw "not a number";
    }
    return (x>0)?x:-x;
}

>abs(-10);
>x:-10
>arguments:-10
//传入不期望的参数个数时
>abs(1,2,3,4,5);
>x:1
>arguments:1
>arguments:2
>arguments:3
>arguments:4
>arguments:5

rest

}
