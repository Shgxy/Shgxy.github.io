Java_注解与反射

Day_01

注解

基本概念

作用：

  • 可以被其他程序（如：编译器）读取；

格式：

@SupressWarnings(value="unchecked")

使用范围：

  • 可以附加在package、class、method、field等上面，相当于给他们添加了额外的辅助信息，还可以通过反射机制编程实现对这些元数据的访问。

内置注解

  • @Override：重写注解，可检验函数名是否出错；
  • @Deprecated：表示不推荐使用，但是可以使用；
  • @SupressWarnings：用来抑制编译时的警告信息，需要填写已定义好的一个或多个参数（“all”、“unchecked”……）。

元注解

用于注解其他注解 ，Java中定义了4个

  • @Target：描述注解的使用范围；💡
  • @Retention：用于描述注解的生命周期（SOURCE<CLASS<RUNTIME）；💡
  • @Documented：说明注解包含在 javadoc 中；
  • @Inherited：子类可以继承父类的注解。
//定义一个注解
//@Target：表示该注解可以用在哪些地方
@Target(value = {ElementType.METHOD,ElementType.TYPE})
//@Retention：表示该注解在什么地方还有效
//RUNTIME>CLASS>SOURCES，用的最多的是RUNTIME
@Retention(value = RetentionPolicy.RUNTIME)
//@Documented：表示是否将该注解生成在JAVADoc中
@Documented
//@Inherited：表示子类可以继承父类的注解
@Inherited
@interface MyAnnotation{//在同一文件下不能加public，内部类
}

自定义注解

格式

  • public @interface 注解名 { 定义内容 }

参数：

  • 可以使用 default 来声明参数的默认值；
  • 如果只有一个参数成员，一般参数名为 value；
  • 如果没有默认值，必须给注解赋值

反射

静态、动态语言

动态语言：

  • 运行时可以改变其结构的语言。

  • 主要的动态语言：

    • Object-C、C#、JavaScript、PHP、Python等。

静态语言：

  • 运行时结构不可变。

  • 主要的静态语言：

    • Java、C、C++。

Java不是动态语言，但Java具有一定的”动态性”,可以利用反射机制获得类似动态语言的特性。

概述

正常方式：引入需要的”包类”名称 -> 通过 new 实例化 -> 获取实例化对象；

反射方式：实例化对象 -> getClass() 方法 -> 得到完整”包类”名称

public class Test02 {
    public static void main(String[] args) throws ClassNotFoundException {
        //通过反射获取类的class对象

        //一个类在内存中只有一个class对象
        //一个类被加载后，类的整个结构都会被封装在class对象中
        System.out.println(c1.hashCode());//2083562754
        System.out.println(c2.hashCode());//2083562754

    }
}

Class 类

通过对象可以得到：

  • 某个类的属性、方法、构造器，以及实现了哪些接口。

此外：

  • Class本身也是一个类；
  • Class对象只能由系统建立对象；
  • 每个实例都会记得自己是由哪个Class实例所生成;
  • 通过Class可以完整的得到一个类中的所有被加载的结构。

获取 Class 类的实例

package com.shgxy.reflection;

public class Test03 {
    public static void main(String[] args) throws ClassNotFoundException {
        Person person = new Student();
        System.out.println("这个人是"+person.name);

        //方式一：通过对象获得
        Class c1 = person.getClass();
        System.out.println(c1.hashCode());//1239731077
        
        //方式二：通过forName获得
        Class c2 = Class.forName("com.shgxy.reflection.Student");
        System.out.println(c2.hashCode());//1239731077
        
        //方式三：通过类名.class获得
        Class c3 = Student.class;
        System.out.println(c3.hashCode());//1239731077
        
        //方式四：基本内置类型的包装类都有一个 Type 属性
        Class c4 = Integer.TYPE;
        System.out.println(c4);//输出：int
        
        //获得父类类型
        Class c5 = c1.getSuperclass();
        System.out.println(c5);//class com.shgxy.reflection.Person
    }
}

Java 内存分析

堆
  • 存放 new 的对象和数组；
  • 可以被所有的线程共享，不会存放别的对象引用。
栈
  • 存放基本变量类型（会包含这个基本变量的具体数值）；
  • 引用对象的变量（会存放这个引用在堆里面的具体地址）。
方法区
  • 可以被所有的线程共享；
  • 包含了所有的 class 和 static 变量。

类何时初始化？

  • 类的主动引用(一定会初始化)
    • JVM启动，先初始化main()所在的类；
    • new出一个对象；
    • 调用类的静态成员(除了final常量)和静态方法；
    • 使用java.lang.reflect包的方法对类进行反射调用；
    • 初始化一个类时，若其父类没有初始化，则会初始化其父类。
  • 类的被动引用(不会发生初始化)
    • 访问静态域，只有真正声明这个域的类才会被初始化。如通过子类引用父类的静态变量时，不会导致子类初始化；
    • 通过数组定义类引用，不会触发此类的初始化；
    • 引用常量不会触发此类的初始化(常量在链接阶段就存到常量池了)。

类加载器

作用：

class文件字节码内容加载到内存中，并将这些静态数据转换成方法区的运行时数据结构，然后在堆中生成一个代表该类的java.lang.Class对象，作为方法区中类数据的访问入口。

package com.shgxy.reflection;

public class Test07 {
    public static void main(String[] args) throws ClassNotFoundException {
        //获取系统类的加载器
        System.out.println(ClassLoader.getSystemClassLoader());
        //获取系统类加载器的父类加载器-->扩展类加载器
        System.out.println(ClassLoader.getSystemClassLoader().getParent());
        //获取系统类加载器的父类加载器-->根加载器（由C/C++编写，读取不到）
        System.out.println(ClassLoader.getSystemClassLoader().getParent().getParent());
        //测试当前类是哪个加载器加载的
        System.out.println(Class.forName("com.shgxy.reflection.Test07").getClassLoader());
        //测试JDK内置的类是谁加载
        System.out.println(Class.forName("java.lang.Object").getClassLoader());
        //如何获得系统类加载器可以加载的路径
        System.out.println(System.getProperty("java.class.lang"));
    }
}

package com.shgxy.reflection;

        System.out.println("-------------------------");

        //获取 类的属性
        Field[] fields = c1.getFields();    //只能找到public
        for (Field field:fields) {
            System.out.println(field);
        }
        System.out.println("-------------------------");
        fields = c1.getDeclaredFields();    //能找到全部的属性
        for (Field field:fields) {
            System.out.println(field);
        }

        //获取指定属性的值
        Field name = c1.getDeclaredField("name");
        System.out.println(name);
        System.out.println("-------------------------");

        //获得类的方法
        Method[] methods = c1.getMethods(); //获得本类及其父类的全部 public 方法
        for (Method method : methods) {
            System.out.println("正常的："+method);
        }
        System.out.println("-------------------------");
        methods = c1.getDeclaredMethods();  //获得本类的所有方法
        for (Method method : methods) {
            System.out.println("getDeclaredMethods："+method);
        }
        System.out.println("-------------------------");

        //获取指定方法
        Method getName = c1.getMethod("getName", null);
        Method setName = c1.getMethod("setName", String.class);
        System.out.println(getName);
        System.out.println(setName);
        System.out.println("-------------------------");

        //获取指定的构造器

有了 Class 对象，能做什么？

// 获得 Class 对象
Class c1 = Class.forName("com.shgxy.reflection.User");

创建类的对象：调用 Class 对象的 newInstance() 方法

  • 类必须有一个无参构造器；
  • 类的构造器的访问权限需要足够。
// 构造一个对象
User user = (User)c1.newInstance();//本质是调用了类的无参构造器
System.out.println(user);

输出：User{name='null', id=0, age=0}

如果删除准备文件一 User 中的无参构造器，则上述语句运行报错。

或者不采用无参构造器，采用以下方法:

// 通过构造器创建对象
Constructor constructor = c1.getDeclaredConstructor(String.class, int.class, int.class);
User user2 = (User)constructor.newInstance("sxy",1,18);
System.out.println(user2);

输出：User{name='sxy', id=1, age=18}

调用指定的方法
//通过反射调用普通方法
User user3 = (User) c1.newInstance();
//通过反射获取一个方法
Method setId = c1.getDeclaredMethod("setId", int.class);

//invoke：激活的意思
//参数（对象，"方法的值"）
setId.invoke(user3,10086);
System.out.println(user3.getId());
System.out.println("------------------------");
//通过反射来操作属性
User user4 = (User) c1.newInstance();
Field id = c1.getDeclaredField("id");
//不能直接通过操作私有属性，我们需要关闭程序的安全检测，属性或者方法的setAccessible(true)。
id.setAccessible(true);
id.set(user4,10010);
System.out.println(user4.getId());

输出：

10086

------------------------

10010

setAccessible()
  • Method/Constructor/Field都有这个方法
  • 用于启动或关闭访问安全检查
  • 可以访问私有成员

性能对比分析

package com.shgxy.reflection;

import java.lang.reflect.InvocationTargetException;
import java.lang.reflect.Method;

public class Test10 {
    //普通方式调用
    public static void test1(){
        User user = new User();
        long startTime = System.currentTimeMillis();

        for (int i = 0; i < 1000_000_000; i++) {
            user.getId();
        }

        long endTime = System.currentTimeMillis();

        System.out.println("普通方式调用："+(endTime-startTime)+"ms");
    }
    //反射方式调用
    public static void test2() throws NoSuchMethodException, InvocationTargetException, IllegalAccessException {
        User user = new User();
        Class c1 = user.getClass();
        Method getId = c1.getDeclaredMethod("getId",null);
        long startTime = System.currentTimeMillis();

        for (int i = 0; i < 1000_000_000; i++) {
            getId.invoke(user,null);
        }

        long endTime = System.currentTimeMillis();

        System.out.println("反射方式调用："+(endTime-startTime)+"ms");
    }
    //反射方式调用 关闭检测setAccessible(true)
    public static void test3() throws NoSuchMethodException, InvocationTargetException, IllegalAccessException {
        User user = new User();
        Class c1 = user.getClass();
        Method getId = c1.getDeclaredMethod("getId",null);
        getId.setAccessible(true);
        long startTime = System.currentTimeMillis();
        for (int i = 0; i < 1000_000_000; i++) {
            getId.invoke(user,null);
        }
        long endTime = System.currentTimeMillis();

        System.out.println("关闭检测方式调用："+(endTime-startTime)+"ms");
    }

    public static void main(String[] args) throws NoSuchMethodException, IllegalAccessException, InvocationTargetException {
        test1();
        test2();
        test3();
    }
}

/*输出：
* 普通方式调用：3ms
* 反射方式调用：1814ms
* 关闭检测方式调用：1243ms
* */

Java中的泛型是为了在javac时确保数据安全性的同时免去强制类型转换的问题，编译完成后，和泛型相关的类型将被擦除

public class Test11 {
    public void test01(Map<String,User> map, List<User> list){
        System.out.println("test01");
    }
    public Map<String,User> test02(){
        System.out.println("test02");
        return null;
    }

    public static void main(String[] args) throws NoSuchMethodException {
        Method method = Test11.class.getMethod("test01", Map.class, List.class);
        for (Type genericParameterType : method.getGenericParameterTypes()) {
//            System.out.println(genericParameterType);
            if(genericParameterType instanceof ParameterizedType){
                Type[] actualTypeArguments = ((ParameterizedType) genericParameterType).getActualTypeArguments();
                for (Type actualTypeArgument : actualTypeArguments) {
                    System.out.println(actualTypeArgument);
                }
            }
        }
        //获取返回值信息
        Method method2 = Test11.class.getMethod("test02",null);
        Type genericReturnType = method2.getGenericReturnType();
        if(genericReturnType instanceof ParameterizedType){
            Type[] actualTypeArguments = ((ParameterizedType) genericReturnType).getActualTypeArguments();
            for (Type actualTypeArgument : actualTypeArguments) {
                System.out.println(actualTypeArgument);
            }
        }
    }
}

小练习：ORM

利用注射和反射完成类和表结构的映射关系，具体如下。

  • 类↔表
  • 属性↔字段
  • 对象↔记录
package com.shgxy.reflection;

import java.lang.annotation.*;

public class Test12 {
    public static void main(String[] args) throws ClassNotFoundException, NoSuchFieldException {
        Class c1 = Class.forName("com.shgxy.reflection.Student2");
        
        //通过反射获得注解
        Annotation[] annotations = c1.getAnnotations();
        for (Annotation annotation : annotations) {
            System.out.println(annotation);
        }

        //获得注解的value值
        TableGod tableGod = (TableGod)c1.getAnnotation(TableGod.class);
        String value = tableGod.value();
        System.out.println(value);

        //获得类指定的注解
        java.lang.reflect.Field f = c1.getDeclaredField("id");
        Field annotation = f.getAnnotation(Field.class);
        System.out.println(annotation.columnName());
        System.out.println(annotation.type());
        System.out.println(annotation.length());
    }
}
@TableGod("db_student")
class Student2{

    @Field(columnName = "db_id",type = "int",length = 10)
    private int id;
    @Field(columnName = "db_age",type = "int",length = 10)
    private int age;
    @Field(columnName = "db_name",type = "varchar",length = 10)
    private String name;

    public Student2(){}

    public Student2(int id, int age, String name) {
        this.id = id;
        this.age = age;
        this.name = name;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    @Override
    public String toString() {
        return "Student2{" +
                "id=" + id +
                ", age=" + age +
                ", name='" + name + '\'' +
                '}';
    }
}

//类名注解
@Target(ElementType.TYPE)
@Retention(RetentionPolicy.RUNTIME)
@interface TableGod{
    String value();
}
//属性注解
@Target(ElementType.FIELD)
@Retention(RetentionPolicy.RUNTIME)
@interface Field{
    String columnName();
    String type();
    int length();
}
