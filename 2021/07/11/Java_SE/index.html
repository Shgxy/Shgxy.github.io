继承

关键词：extends，扩展。无法继承 final 修饰的类。

在 Java 中，所有的类，都默认直接或者间接继承 Object 类。

在 Java 中，只有单继承，没有多继承，即只能继承一个父类。

但一个父类可以有很多子类。

package com;

import com.opp.demo06.Student;

    public static void main(String[] args) {
        
        Student s1 = new Student();
        Person  s2 = new Student();
        Object  s3 = new Student();

        //对象能执行哪些方法，主要看对象左边的类型，和右边关系不大

    }

instanceof

（类型转换）引用类型；

instanceof是Java中的二元运算符，左边是对象，右边是类；当对象是右边类或子类所创建对象时，返回true；否则，返回false。

System.out.println(X instanceof Y);//true：实例X 与 类Y 之间是父子关系
Student s1 = new Student();

Object  s3 = new Student();

  System.out.println(p1 instanceof Student);	//false
  System.out.println(s2 instanceof Student);	//true
  System.out.println(s3 instanceof Student);	//true
  System.out.println(s1 instanceof Person);		//true
  System.out.println(s2 instanceof Person);		//true
  System.out.println(s2 instanceof Object);		//true

Error和Exception

IO流没讲

//	异常分为两大类：错误Error和异常Exception
//	定义如下基类，作为所有异常的超类
java.lang.Throwable;

Error

由java虚拟机生成并抛出，大多数错误与代码编写者所执行的操作无关。

Exception

一个重要子类 RuntimeException (运行时异常)

这些异常一般是由程序逻辑错误引起的。

两者区别

Error是灾难性的错误，程序一般无法控制和处理，出现这种错误JVM一般会选择终止线程；

Exception通常情况下是可以被程序处理的，并且在程序中应该尽可能的去处理这些异常。

异常处理机制

五个关键字

try catch finnally throw throws

抛出、捕获

package com.opp.demo11;

public class Test {
    public static void main(String[] args) {
        int a = 1;
        int b = 1;
        try {//    try内监控区域
            System.out.println(a/b);// 想要捕获的异常类型，最高的是Throwable
        }catch (ArithmeticException e){//   捕获异常
            System.out.println("ArithmeticException异常");
        }finally {//    无论程序出不出错，都会运行，可以不用该语句
            System.out.println("原式为："+a+"/"+b);
        }

    }
}

catch可以多级使用，但是要保证最大的异常（如Throwable）放在最下面。

快捷键

  1. 选中 想要捕获异常的语句；

  2. crtl + alt + T ；

  3. 选择 “try/catch/finally”
